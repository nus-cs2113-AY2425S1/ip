Hello, I am Diana! A personal assistant that helps you keep track of your tasks :)
Here are the list of things that Diana can do for you
1. Create a todo task eg. [todo read book]
2. Create an event task eg. [event read book /from 2pm /to 4pm]
3. Create a deadline task eg. [deadline read book /by 2pm]
4. Type either mark or unmark to indicate the completion rate of your task
5. Type list to view the your list of tasks.
6. Type bye to exit the programme
